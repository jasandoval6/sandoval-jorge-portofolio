## Overview

El Paso Active Engage is a smart city application concept designed to support older adults through activity engagement, sponsorship, and point-based incentives. 
This project focused on **requirements elicitation, system modeling, and architectural design**, rather than implementation.

The goal was to translate stakeholder needs into a clear, scalable system design suitable for future development.

---

## Key Contributions

- Conducted stakeholder interviews and requirements elicitation
- Defined system actors, responsibilities, and interactions
- Designed CRC cards to clarify object responsibilities and collaborations
- Created context diagrams to model system boundaries and external entities
- Developed Level 1 Data Flow Diagrams to represent core system processes
- Addressed accessibility, role-based access, and operational considerations
